Introduction

This is a color utility originally created to convert between RGB and HSV values that has expanded into a tool for selecting colors from a variety of sources. It was created by Scott Mitting in 2024 to save time at work when needing to pick colors sometimes when I don't have Photoshop available.

Upcoming Features

This section lists the features that will be added in the future. They were originally intended to be added to the initial release, but if they are still listed here they either didn't get finished in time or this documentation needs updated.

The progress on these features, as well as requests for other new features, can be managed on the SUColorKit Bug Tracker

  • Import/Export Allowing users to copy sets of files into and out of the virtual file system, so they can make backups of their settings, and share images and palettes with their friends and colleagues.
  • Palette History / Saved Colors The ability to have a list of colors that have been recently selected within the tool, and to save them to use later.
  • Drag to Select The ability to select multiple colors by dragging the mouse across the image, sorting the colors by the most common colors, and grouping colors together than are equal within a tolerance level.
  • Grab Pixel from Web Page The user will be able to use an eyedropper tool to grab colors off of the currently visible webpage.
  • Multi-Color and Palette Management Allowing for multiple colors to be placed into a set, with tools to help choose new colors that fit the colors in the palette so far.
  • External Tool Integration For example, I would like users to be able to search and import palettes from websites such as https://lospec.com/palette-list in a way that makes sure those sources get credit for the results.



Usage

The following sections describe how to use the tool, broken up by each tab. You can drag the title bar to turn the window into a separate browser window, which can be more flexible than the dropdown provided by the extension.

The three-bars icon is a drop-down menu providing an alternative method of navigating to common functions in the app, e.g. Upload Files is available from this menu.




Usage - Gradients

The Gradients screen allows the user to select colors from a 2D gradient which changes against a third value (similar to Photoshop) and by adjusting sliders of both RGB and HSV values. Both the RGB and HSV sliders can be adjusted independently while they are automatically kept in sync, converting between RGB and HSV values.

The type of gradient to use can be selected from the palette dropdown, which allows for all permutations of Hue/Sat/Val (HSV) and Red/Green/Blue (RGB) to be selected from a gradient, as well as providing palette samples of classic computers such as VGA256. You can add additional gradients by using the File Manager under the Settings tab.

The color selection is displayed in a box in the upper-left corner of the view, optionally with an overlay image for comparison. The hex color is displayed under the color selection, with a button to copy its value to the browser. This feature will be greatly expanded in the future.

A checkbox is available to control where an overlay image is displayed on top of the selected color to easily compare in real time your color selection. The image to use for this is customized under the Settings tab in the File Manager.




Usage - Named Colors

Named colors shows a list of color names, their hex values, and a sample swatch of each color from a list that can be scrolled vertically. Above the list is a drop-down of palette choices providing different sets of named colors. Additional lists can be added by the user using the File Manager in the Settings tab.

The selected color is displayed in the upper-left just like the Gradients tab.




Usage - Photos

The photos view allows colors to be selected by using an eyedropper tool on one of the photos saved in the File Manager. Users can add additional photos using the File Manager in the Settings tab.

The current photo can be moved by using the arrow keys, and can be zoomed by using the page-up and page-down keys.




Usage - Settings

The settings tab allows access to the file manager, which a text description of the folders within the virtual file system, and how many of each are currently available.

At the bottom is a dropdown where you can choose which photo to use as an overlay image over the selected color. You can upload your own images into the Overlay Images folder.
